- Fact tables
- Dimension tables
- Star schemas
- Data marts
- ML feature tables
- Production-ready ML models

------------------------------------------------------------
3. Fact Tables
fact_policies – premium, tenure, renewal outcomes
fact_members – demographic & risk attributes
fact_claims – payout, claim amounts, fraud & high-cost flags

------------------------------------------------------------
4. Dimensions
dim_channel – marketing/sales channel
dim_product_line – type of insurance product
dim_providers – fraud flags, risk tier for providers

------------------------------------------------------------
5. Star Schemas
star_policies = fact_policies + dim_channel + dim_product_line
star_members  = fact_members  + dim tables
star_claims   = fact_claims   + dim_providers

------------------------------------------------------------
6. Data Marts
dm_policy_retention – renewal KPIs
dm_member_value – customer lifetime value
dm_claims_experience – cost ratios, risk scoring, provider analysis

------------------------------------------------------------
7. ML Feature Tables
ft_policy_churn – engineered features for churn prediction
ft_claims_risk – fraud/high-cost classification features

------------------------------------------------------------
8. Model Storage
policy_churn_model – logistic regression pipeline
claims_risk_model – logistic regression pipeline

------------------------------------------------------------
9. Architecture Explanation
Bronze → raw landing  
Silver → validated data with DQ rules  
Gold → analytic models (facts, dims, stars, marts, ML features)

------------------------------------------------------------
